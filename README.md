# WhatsApp Bot in Go

A complete, production-ready WhatsApp Bot Client using the `whatsmeow` library, structured around Clean Architecture principles.

## Prerequisites
- Go 1.22+
- Optional: PostgreSQL (for cloud/multi-instance deployments)
- `make` utility

## Quick Start

1. **Clone & Configure**
   Copy `config/config.yaml` and adjust settings (like `admin_jids`).
   By default, it uses SQLite which stores the session internally in `data/session.db`.

2. **Run the Bot**
   ```bash
   make tidy
   make run
   ```

3. **Scan QR Code**
   On the first run, a QR code will be printed to the terminal. Open WhatsApp on your phone, go to **Linked Devices**, and scan the QR code.
   On subsequent runs, the session is resumed automatically.

## Adding a new Command
1. Create a new file in `internal/command/`, e.g., `hello.go`.
2. Embed `BaseCommand` to inherit default methods:
   ```go
   type HelloCommand struct {
       BaseCommand
   }
   func (c *HelloCommand) Name() string { return "hello" }
   func (c *HelloCommand) Description() string { return "Says hello" }
   func (c *HelloCommand) Execute(ctx *Context) error {
       return ctx.Reply("Hello world!")
   }
   ```
3. Register it in `cmd/bot/main.go`:
   ```go
   registry.Register(&command.HelloCommand{})
   ```

## Adding a new Guard
1. Implement the `Guard` interface in `internal/guard/`.
2. Attach it to your command:
   ```go
   func (c *MyAdminCommand) Guards() []guard.Guard {
       return []guard.Guard{guard.NewAdminGuard(admins)}
   }
   ```

## Switching from SQLite to PostgreSQL
1. Ensure your PostgreSQL instance is running.
2. Edit `config.yaml`:
   ```yaml
   session:
     driver: "postgres"
     postgres:
       dsn: "postgres://user:pass@localhost:5432/whatsbot?sslmode=disable"
   ```

## Environment Variables Reference
You can override any config.yaml key via environment variables:
- `BOT_COMMAND_PREFIX` (e.g., set to `/`)
- `BOT_SESSION_DRIVER` (e.g., set to `postgres`)
- `BOT_SESSION_POSTGRES_DSN`
- `BOT_LOG_LEVEL` (`debug`, `info`, `warn`, `error`)
